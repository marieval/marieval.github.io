About Me

I´m a self-taught web-developer having previous working experience as a researcher, university teacher and a program manager. Approx. in 2017 I started learning web development and now I devote as much time as possible to learning new things in this area.

I started learning through various online materials (CodeCademy, FreeCodeCamp) and later attended several in-person workshops (SQL, Bootstrap, Inkscape, Web-Design In Practice and others) and 3-month courses (Java, JavaScript). In August 2018 I discovered some great courses on Udemy and since then I finished 5 courses there (on CSS, JavaScript and Web Development) and currently am doing 3 more there (on Node.js, React and Advanced Web Development).

I got inspired by 100-Days-Of-Code and No-Zero-Days challenges and started with my version in August 2018. Now I´m far behind Day#100 and keep going. You can see my progress on my GitHub.

  • React

  • Bootstrap, Tachyons
  • Git & GitHub
  • Command Line
